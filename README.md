Project Title = AI contact manager

######Description
This project is a full-stack Contact Management System using Spring Boot for the backend and Next.js for the frontend, supporting role-based authentication. Users can add, edit, and manage contacts with details like name, city, company, and job title. The system integrates AI moderation to flag misleading or suspicious contact data automatically. Admins have a dashboard to view users, delete accounts, and monitor AI-flagged contacts. It demonstrates secure authentication, AI-powered moderation, and full-stack CRUD operations in a real-world scenario
.
#####Features

1. **Full-Stack Architecture**

   * Backend built with **Spring Boot**
   * Frontend built with **Next.js**

2. **Role-Based Authentication**

   * Separate **user** and **admin** roles
   * Secure login and registration system

3. **Contact Management (CRUD Operations)**

   * Add, edit, view, and delete contacts
   * Each contact includes details like **name, city, company, and job title**

4. **AI-Powered Moderation**

   * Automatically flags **misleading or suspicious contact data**
   * Helps maintain data quality and security

5. **Admin Dashboard**

   * View all registered users
   * Delete user accounts
   * Monitor AI-flagged contacts

6. **Real-World Use Case Implementation**

   * Demonstrates **secure authentication**, **role-based access**, and **full-stack CRUD** operations
   * Suitable for learning **enterprise-grade full-stack development**

7. **Responsive Frontend**

   * User-friendly interface built with Next.js
   * Works across devices and screen sizes
